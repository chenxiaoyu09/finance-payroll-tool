# 财务算薪系统线上环境变量模板

## 后端生产环境示例

```env
NODE_ENV=production
APP_PORT=3000
CORS_ORIGINS=https://payroll.your-domain.com

DB_HOST=your-mysql-host
DB_PORT=3306
DB_NAME=finance_payroll_prod
DB_USER=your_db_user
DB_PASSWORD=your_db_password
DATABASE_URL=mysql://your_db_user:your_db_password@your-mysql-host:3306/finance_payroll_prod

REDIS_HOST=your-redis-host
REDIS_PORT=6379
REDIS_PASSWORD=

UPLOAD_DIR=./uploads

JWT_SECRET=replace_with_long_random_secret
JWT_EXPIRES_IN=7d
```

## 前端生产环境示例

```env
VITE_API_BASE_URL=https://api.your-domain.com/api
```

## 本地开发环境示例

```env
NODE_ENV=development
APP_PORT=3000
CORS_ORIGINS=http://127.0.0.1:5173,http://localhost:5173
VITE_API_BASE_URL=http://127.0.0.1:3000/api
DATABASE_URL=mysql://root:123456@127.0.0.1:3306/finance_payroll_dev
UPLOAD_DIR=./uploads
JWT_SECRET=replace_me_with_a_secure_secret
JWT_EXPIRES_IN=7d
```
