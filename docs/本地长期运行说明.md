# 本地长期运行说明

## 适用场景

适合你现在这个阶段：

- 暂时不上线
- 主要还是你自己用
- 需要本地稳定反复开发和测试

## 推荐本地配置

`.env` 建议保持：

```env
APP_HOST=127.0.0.1
APP_PORT=3000
CORS_ORIGINS=http://127.0.0.1:5173,http://localhost:5173
VITE_DEV_HOST=127.0.0.1
VITE_API_BASE_URL=http://127.0.0.1:3000/api
```

## 启动方式

后端：

```bash
cd apps/server
pnpm dev
```

前端：

```bash
cd apps/web
pnpm dev
```

## 访问地址

- 前端：`http://127.0.0.1:5173`
- 后端：`http://127.0.0.1:3000/api`

## 优点

- 最稳
- 最简单
- 不用考虑网络和跨设备问题
- 改代码后反馈最快

## 适合做什么

- 算薪逻辑开发
- Excel 导入规则调试
- 导出文件校验
- 页面交互细节打磨
