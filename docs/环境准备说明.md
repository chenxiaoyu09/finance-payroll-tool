# 环境准备说明

## 1. 当前阶段目标

当前阶段只做开发环境准备，确保后续可以顺利进入算薪工具编码阶段。

## 2. 必备软件

建议安装或确认以下软件：

- `VS Code` 或 `Cursor`
- `Git`
- `Node.js 22+`
- `pnpm 10+`
- `MySQL 8`
- `Redis`
- `DBeaver` 或 `Navicat`
- `Apifox` 或 `Postman`

## 3. 当前机器检查结果

已确认：

- `Node.js v22.14.0`
- `npm 10.9.2`
- `pnpm 10.6.3`
- `Git 2.33.1`
- `MySQL 8.0.28`

未确认或原先缺失：

- `Redis`
- `Docker`

说明：

- `Docker` 不是必须项
- 当前项目完全可以先直接跑本机 `MySQL + Redis`

## 4. 本地建议服务配置

### MySQL

建议创建数据库：

- `finance_payroll_dev`

编码建议：

- `utf8mb4`
- `utf8mb4_unicode_ci`

### Redis

默认建议本地配置：

- Host：`127.0.0.1`
- Port：`6379`

## 5. 项目建议技术栈

### 后端

- `NestJS`
- `TypeScript`
- `Prisma`
- `exceljs`
- `class-validator`

### 前端

- `Vue 3`
- `TypeScript`
- `Vite`
- `Element Plus`
- `Pinia`
